# **App Name**: Consciousness Class

## Core Features:

- User Roles: User authentication and role management (Superadmin, Creator, Student).
- Course Listings: Course and membership listing and detail pages.
- Content Delivery: Integrated video/audio/PDF delivery.
- Marketing & CRM: Email marketing, AI Chatbot Assistants, Smart Journaling, WhatsApp Integration (Roadmap Features).
- Payment Gateway: Integration with Stripe.

## Style Guidelines (Natureza Palette):

- Primary color: Olive Green / Sage (Evokes grounding and natural healing).
- Background color: Bone / Off-White (Warm and readable).
- Form/Foreground format: Charcoal / Soft Black (Premium elegance).
- Accent color: Terracotta / Clay.
- Body font: iOS San Francisco (Inter / system-ui).
- UI Elements: Apple HIG compliance with 'ios-list' patterns and organic corners.
